# 🏡 Airbnb Dashboard

## 📌 Project Overview

This project presents an interactive **Airbnb Dashboard** developed using **Power BI** to analyze Airbnb listings, pricing, availability, and customer ratings. The dashboard helps identify market trends, property performance, and customer preferences through interactive visualizations.

---

## 🎯 Objectives

- Analyze Airbnb listing performance.
- Compare prices across different locations.
- Identify highly rated and popular properties.
- Monitor availability and occupancy trends.
- Provide business insights through interactive dashboards.

---

## 📊 Key Performance Indicators (KPIs)

- Total Listings
- Average Price
- Average Rating
- Total Reviews
- Availability
- Host Count

---

## 📈 Dashboard Features

- Price Analysis by Location
- Property Type Distribution
- Average Rating Analysis
- Availability Trends
- Top Hosts Analysis
- Interactive Filters and Slicers
- Dynamic Charts and KPI Cards

---

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX
- Microsoft Excel
- SQL

## 📊 Business Insights

- Identified locations with the highest average prices.
- Compared property types based on customer ratings.
- Analyzed listing availability across different regions.
- Evaluated host performance using reviews and ratings.
- Created an interactive dashboard to support data-driven decision-making.
